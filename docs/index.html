Before you begin

This is a cross-platform class which uses Xamarin.Forms to to target iOS, Android, or Windows. To complete the coding exercises, you will need a Xamarin development environment with either an emulator/simulator or a device to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

This class assumes you have already set up an Azure account and have a basic knowledge of the Azure management portal. If you have not done anything with Azure yet, then please review the materials from AZR101 - Introduction to Azure.

Overview

This class dives into the authentication aspects of Azure Mobile applications. We will examine how to enable authentication in the Azure portal, what impact that has on your service, and then how to properly connect and authorize your application to continue to utilize the APIs of your Azure App Service.

Lecture

AZR120 - Authentication with Azure App Services.

Objectives

  1. Introduce authentication options
  2. Require authentication in an Azure App Service
  3. Log in to your mobile service
  4. Add support for multiple users

Exercises
